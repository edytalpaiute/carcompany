# Frontend Next.js Skill: shadcn + Tailwind

## Name
frontend-nextjs-shadcn-tailwind

## Description
Expert skill for building modern frontend experiences in Next.js using shadcn UI and Tailwind CSS.

## Capabilities
- Scaffold and refine `app` router pages, layouts, and API routes in Next.js 16+
- Create responsive UIs with Tailwind CSS v4 utility classes
- Generate and customize shadcn-compatible components using `components.json`
- Integrate Radix UI primitives and accessible interaction patterns
- Implement design tokens, themes, dark mode, and animation with Tailwind and shadcn utilities
- Optimize frontend code for performance, maintainability, and developer ergonomics

## Project Context
This repository uses:
- `next` 16.2.6 with the `app` directory
- `tailwindcss` v4 and `@tailwindcss/postcss`
- `shadcn` 4.x and `radix-ui`
- component aliases such as `@/components`, `@/lib`, and `@/components/ui`

## Typical Tasks
- Build new pages with shadcn component design patterns
- Add reusable UI components and variant systems
- Convert plain JSX into accessible, themeable Tailwind/shadcn components
- Wire up forms, modals, tabs, cards, navigations, and responsive layout
- Keep `app/globals.css` aligned with Tailwind and shadcn CSS setup

## Example Prompts
- "Create a responsive hero section using shadcn buttons and Tailwind utilities."
- "Implement a dashboard card grid with dark mode support and accessible keyboard navigation."
- "Add a shadcn-styled login form that uses Tailwind spacing, form controls, and error states."

## Notes
This skill is tailored for this repository's current setup and should be used when working on frontend UI improvements, component creation, and styling within the existing Next.js + shadcn + Tailwind stack.
